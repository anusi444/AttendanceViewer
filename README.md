# 📊 Attendance Viewer System

A simple and efficient web-based application to manage and view student attendance.
Built using Java (JSP & Servlet), Hibernate, and MySQL.

---

## 🚀 Features

✨ Add and manage student records
📅 Mark and track attendance
📊 View attendance details easily
🗂️ Organized dashboard for better usability
🔐 Secure login system

---

## 🛠️ Tech Stack

* 💻 Java (JSP & Servlets)
* 🗄️ MySQL Database
* 🔗 Hibernate (ORM Framework)
* 🌐 HTML, CSS, JSP
* ⚙️ Apache Tomcat Server

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/AttendanceViewer.git
```

2. Open in IntelliJ / Eclipse

3. Configure MySQL database

4. Update database credentials in:

```
Hibernate.cfg.xml
```

5. Run on Apache Tomcat Server
